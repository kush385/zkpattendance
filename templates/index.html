How it works

Protocol : Schnorr Identification (Sigma Protocol) Params : p = 23 (prime), g = 5 (generator), ord(g) = 22 ── Registration ──────────────────────────────────── Student chooses secret x ∈ [2, 21] (random) System computes public y = g^x mod p Stored on server: y (never x) ── Attendance (Schnorr Proof) ────────────────────── Teacher broadcasts c (random challenge / nonce) Student picks random r ∈ [2, 21] Student computes: t = g^r mod p s = (r + c·x) mod 22 Sends (t, s) — no x! ── Verification ──────────────────────────────────── Server checks: g^s ≡ t · y^c (mod p) Valid ⟹ attendance marked ✓
